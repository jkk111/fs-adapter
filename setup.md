# windows
Prerequisites
Node 8.x
Python 2.7
Visual Studio
Net Framework 2.0
Net Framework 2.0 SDK

Run the following
```
npm install
```

# ubuntu
Run the following commands
```
curl -sL https://deb.nodesource.com/setup_8.x | sudo -E bash -
sudo apt-get install -y nodejs
npm install
```

To Run specify your target host in config.json
then run the following
```
node index.js
```
